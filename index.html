Описание

Псевдо-класс :link используется для выбора непосещённых ссылок (непосещенная ссылка - ссылка, по которой еще не был осуществлен переход).

Обычно он используется в паре с псевдо-классом :visited (определяет стиль для посещённых ссылок), так как без него стили могут работать не так как задумано, например, если указать фон для непосещённой ссылки и при этом не указать явно стиль для посещённой ссылки, то фон, заданный для непосещённой ссылки после перехода по ней, всё равно останется.
